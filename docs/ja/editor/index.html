私達の楽譜編集ソフトウェアのヘルプへようこそ！このページからアプリや各種素晴らしい機能の詳細をご覧頂けます。

Flatは、どなたでもご利用頂ける最高の楽譜作成ソフトです。ウェブブラウザや、モバイルデバイス用のアプリとしてご利用頂く事ができ、簡単にアクセスして使えるように美しくデザインされた製品です。 楽譜作成ソフトを初めてご利用される方でも、わずか数分で最初の作品を手掛ける準備が完了します。

Flatには、楽譜の表示、作曲、編曲、共同作業、印刷、エクスポート、共有といった必要な全ての機能が含まれています。MIDIデバイス接続からリアルタイム共同編集や全ての修正履歴の記録など、Flatの画期的で最高の機能があなたの日々の創作活動を支援します！

以下のチュートリアルや各機能についての詳細なヘルプコンテンツをご覧ください。ウェブ上の編集画面を利用中の場合は、画面右上のヘルプボタン "?"をクリックしていつでもこのヘルプを開く事ができます。

手順ごとのチュートリアル
